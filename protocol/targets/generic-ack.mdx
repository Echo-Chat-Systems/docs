# Generic Acknowledge

At many points throughout the application it is useful to have a generic acknowledge message to allow an actor to acknowledge
that they have received and actioned a request without needing to return any data.

## Structure

```json Example
{
    "target": "",
    "data": {
        "action": "",
        "params": {
            "status": "ok"
        }
    }
}
```

```json Schema
}
```

### Parameters

<ParamField body="target" type="string" required>
    Target of the message being acknowledged.
</ParamField>
<ParamField body="action" type="string" required>
