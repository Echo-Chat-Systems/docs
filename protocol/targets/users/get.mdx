# `get`

Retrieves a [**public user**](/protocol/definitions/users/data.mdx#public-user-structure) by their user ID (Ed25519 public key). This action does not require authentication and returns only non-sensitive information.

## Overview

The `get` action allows clients to fetch basic profile information about any user by providing their public signing key. It is intended for public display purposes (e.g., when viewing another user's profile).
