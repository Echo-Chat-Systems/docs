# `users`

The `users` target handles retrieving, updating, and interacting with user profiles and reputation information.
These actions expose both public and private user information, depending on authentication and intent.

---

## Available Actions

| Name                                                 | Description                                                                |
|------------------------------------------------------|----------------------------------------------------------------------------|
| [`get`](/protocol/targets/users/get.mdx)             | Retrieves the public profile of a specific user.                           |
| [`get-self`](/protocol/targets/users/get-self.mdx)   | Retrieves the full authenticated user state and settings.                  |
| [`update`](/protocol/targets/users/update.mdx)       | Updates the current user's profile and status fields.                      |
| [`rep-add`](/protocol/targets/users/rep-add.mdx)     | Adds a reputation remark to a user profile.                                |
| [`rep-del`](/protocol/targets/users/rep-del.mdx)     | Allows a user to delete a reputation remark they have previously made.     |
| [`rep-query`](/protocol/targets/users/rep-query.mdx) | Allows a user to query what reputation remarks they have made in the past. |
