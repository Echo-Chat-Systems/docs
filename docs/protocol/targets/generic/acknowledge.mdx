# Generic Acknowledge

At many points throughout the application it is useful to have a generic acknowledge message to allow an actor to acknowledge
that they have received and actioned a request without needing to return any data.

## Structure

<SchemaExample path="/comms/generic-ack.json">
```json Example
{
    "target": "",
    "data": {
        "action": "",
        "params": {
            "status": "ok"
        }
    }
}
```
</SchemaExample>

<ParamsList>
    <Param name="target" type="string" required>
        Target of the message being acknowledged.
    </Param>
    <Param name="action" type="string" required>
        Action of the message being acknowledged.
    </Param>
</ParamsList>
