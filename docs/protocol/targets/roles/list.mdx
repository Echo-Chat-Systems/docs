# `list`

Lists roles inside a specific guild.

## Overview

Allows any guild member to list all roles available in the guild.

---

## Client → Server

<SchemaExample path="/comms/roles/list.json">
```json Example
{
    "target": "roles",
    "data": {
        "action": "list",
        "params": {
            "guild": 3980959480774269000
        }
    }
}
```
</SchemaExample>

<ParamsList>
    <Param name="guild" type="int" required>
        Guild [id](/docs/protocol/snowflake).
    </Param>
</ParamsList>
