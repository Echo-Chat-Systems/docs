# `create`

Creates a new guild.

## Overview

Allows the user to create a new guild.

---

## Client → Server

<SchemaExample path="/comms/guilds/create.json">
```json Example
{
    "target": "guilds",
    "data": {
        "action": "create",
        "params": {
            "name": "The Storag Vault"
        }
    }
}
```
</SchemaExample>

<ParamsList>
    <Param name="name" type="string" required>
        Initial guild name.
    </Param>
</ParamsList>

## Server → Client

Provided the guild creation succeeds, the user who initiated the `create` is added to the guild after creation.

<SchemaExample path="/comms/guilds/create-response.json">
```json Example
{
    "target": "guilds",
    "data": {
        "action": "create",
        "params": {
            "id": 17204232703081746000
        }
    }
}
```
</SchemaExample>

<ParamsList>
    <Param name="id" type="integer" required>
        Newly created guild [id](/docs/protocol/snowflake).
    </Param>
</ParamsList>
