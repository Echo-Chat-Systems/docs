# `delete`

Deletes an existing guild.

## Overview

Allows a user to delete a guild that they are the owner of.

---

## Client → Server

<SchemaExample path="/comms/guilds/delete.json">
```json Example
{
    "target": "guilds",
    "data": {
        "action": "delete",
        "params": {
            "id": 5655277926790978000
        }
    }
}
```
</SchemaExample>

<ParamsList>
    <Param name="id" type="integer" required>
        Guild [id](/docs/protocol/snowflake).
    </Param>
</ParamsList>

## Server → Client

On success, the server returns a [generic acknowledgment](/docs/protocol/targets/generic/acknowledge).

## Errors

| Message         | Description                                                     | Possible Causes                                                     |
|-----------------|-----------------------------------------------------------------|---------------------------------------------------------------------|
| `missing-perms` | User does not have appropriate permission level to delete guild |                                                                     |

---

## Notes

- This action is irreversible
- The client is expected to implement checks on their end to ensure that guilds are not deleted accidentally.
