# `update`

Updates guild information.

## Overview

Allows a user to update guild information for a guild that they have permission to modify.

---

## Client → Server

```json Example
{
    "target": "guilds",
    "data": {
        "action": "update",
        "params": {
            "guild": {  ...  }
        }
    }
}
```

<ParamsList>
    <Param name="guild" type="object" required>
        [Guild data] being updated.
    </Param>

## Server → Client

On success, the server returns a [generic acknowledgment](/docs/protocol/targets/generic/acknowledge).

## Notes

- Certain guild fields are immutable. Only `config`, `customisation`, and `name` are permitted to be changed in this way.
