# Guild Data

This page defines the representation format of a guild within the context of the protocol specs. A guild is defined as
a collection of [channels](/docs/protocol/definitions/channel/data) and
[users](/docs/protocol/definitions/user/data#public-user-structure).

---

## Guild Structure

<SchemaExample path="/guild/guild.json">
```json Example
{
    "id": 17575072144222058000,
    "name": "The Storag Vault",
    "owner": "RUxSNFkqJSQxRV9MaDJSP00vTXRtVSN7",
    "members": [ ... ],
    "channels": [  ...  ],
    "roles": [  ...  ],
    "media": {
        "stickers": [  ...  ],
        "emojis": [  ...  ]
    },
    "config": {  ...  },
    "customisation": {  ...  }
}
```
</SchemaExample>

<ParamsList>
    <Param name="id" type="integer" required>
        [Guild Id](/docs/protocol/snowflake).
    </Param>
    <Param name="name" type="string" required>
        Guild name.
    </Param>
    <Param name="owner" type="string" required>
        Guild Owner [Id](/docs/protocol/definitions/user/identity).
    </Param>

    <Param name="members" type="array" required>
        Array of [guild members](/docs/protocol/definitions/guild/member).
    </Param>

    <Param name="channels" type="array" required>
        Array of [channels](/docs/protocol/definitions/channel/data) belonging to the guild.
    </Param>

    <Param name="roles" type="array" required>
        Array of [roles](/docs/protocol/definitions/guild/role).
    </Param>

    <Param name="media.stickers" type="array">
        Array of [stickers](/docs/protocol/definitions/media/).
    </Param>

    <Param name="media.emojis" type="array">
        Array of [emojis](/docs/protocol/definitions/media/).
    </Param>

    <Param name="config" type="object">
        Guild [configuration](/docs/protocol/definitions/guild/config).
    </Param>

    <Param name="customisation" type="object">
        Guild [customisation](/docs/protocol/definitions/guild/customisation).
    </Param>
</ParamsList>
