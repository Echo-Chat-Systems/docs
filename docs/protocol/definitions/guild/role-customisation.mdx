# Role Customisation

```json Example
{
    "banner": "8103275619140250000",
    "icon": "14915265854996625000",
    "text-colour": "#c548d5",
    "background-colour": "#00ffd5",
    "css": ""
}
```


<ParamsList>
    <Param name="banner" type="string">
        Either a URL linking to a valid image file or a [snowflake](/docs/protocol/snowflake) referencing a [media](/docs/protocol/definitions/media/) item.
    </Param>
    <Param name="icon" type="string">
        Either a URL linking to a valid image file or a [snowflake](/docs/protocol/snowflake) referencing a [media](/docs/protocol/definitions/media/) item.
    </Param>
    <Param name="text-colour" type="string">
        Role label text colour in hexadecimal format. (\# is optional)
    </Param>
    <Param name="background-colour" type="string">
        Role label background colour in hexadecimal format. (\# is optional)
    </Param>
    <Param name="css" type="string">
        Additional guild-wide css.
    </Param>
</ParamsList>
